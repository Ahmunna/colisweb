# Colisweb Api

Colisweb Api est une application web qui permet de marquer des adresses dans la map.

## Installation

Use NPM or YARN to install the dependencies

```bash
npm install
```
or 
```bash
Yarn
```

## Usage
Launch the Start script to run the server on localhost PORT 3000
```bash
Yarn start
```
or 
```bash
npm start
```
